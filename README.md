## 📌 Project Overview

**JobFlow** is a smart recruitment platform that connects **job seekers**, **companies**, and **administrators** within a single ecosystem.

The application provides structured job posting, application tracking, resume handling, notifications, and AI-supported workflow automation using modern mobile technologies.

---

## 🎯 Key Objectives

* Simplify recruitment management
* Provide organized job application tracking
* Enable real-time communication & updates
* Build scalable mobile architecture using Firebase
* Integrate intelligent automation features

## 🏗️ Project Architecture

```
lib/
│
├── Admin_Side/              → Admin functionalities
├── Authentication/          → Login & signup flow
├── CompanySide/             → Recruiter features
├── UserSide/                → Job seeker modules
├── WelcomePages/            → Onboarding & intro screens
├── Widgets/                 → Reusable UI components
├── services/                → Firebase & backend services
├── firebasenotifications/   → Push notification logic
├── Python/                  → AI / processing scripts
│
├── constants.dart           → App constants
└── main.dart                → Application entry point
```

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```bash
git clone https://github.com/diya2405/jobflow.git
```

### 2️⃣ Install Dependencies

```bash
flutter pub get
```

### 3️⃣ Firebase Configuration

* Create Firebase Project
* Enable Authentication
* Enable Firestore Database
* Enable Storage
* Enable Cloud Messaging
* Add:

    * `google-services.json`
    * `GoogleService-Info.plist`

### 4️⃣ Run Application

```bash
flutter run
```
